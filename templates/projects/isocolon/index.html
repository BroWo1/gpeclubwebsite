Isocolon

Key points: Isocolon is a rhetorical device that uses a parallel structure in a series of phrases or clauses that have the same structure, in order to emphasize the message delivering.

Key Characteristics

Usage Circumstance

Key points:

1. Use isocolon when repeating phrases with the same grammatical structure.


2. Use isocolon when appealing to pathos.


3. Use isocolon when appealing to logos.

Real-world Examples

Example 1:

“Veni, vidi, vici” - Julius Caesar
English Translation: “I came, I saw, I conquered.”

Example 2:

“Ask not what your country can do for you—ask what you can do for your country.” - John F. Kennedy, Inaugural Address

Conclusion

Key points:
If used correctly and effectively, the rhetorical device of isocolon can emphasize the idea delivered to the audience, therefore making the work more persuasive and impactful. Below are some tips for you to use isocolon.

1. Ensure the structure is uniform and rhythmic.

2. Use isocolon in less formal circumstances.

3. Align isocolon with the purpose thoughtfully.
